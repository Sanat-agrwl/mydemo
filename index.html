Indic-CLAP: cross-lingual audio distillation

Demo page. Each of the four audio clips is processed by five models. We report the top-3 zero-shot tags and the top-1 retrieved caption per model. The English CLAP row uses CLAP's own English text encoder over the English label set / English caption pool; all other rows are joint Indic students producing Hindi outputs. Tag pool size: 55 labels · caption retrieval pool size: 2000 captions.

English CLAP— frozen reference: CLAP's own English text encoder
TT-MSE— joint student: text-teacher MSE distillation
AT-MSE— joint student: audio-teacher MSE distillation
InfoNCE (λ=1)— joint student: pure InfoNCE contrastive loss
Hybrid (λ=0.10)— joint student: cosine-text ⊕ InfoNCE-audio hybrid (our method)
